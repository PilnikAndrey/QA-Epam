№1  Тестирование регистрации
    1. Переходим на 21vek.by
    2. Нажимаем кнопку "Войти" в правом верхнем углу.
    3. На появившейся форме Входа, нажимаем кнопку "Регистрация".
    4. На новой форме вводим ваш электронный адрес.
    5. Нажимаем кнопку "Продолжить".
    
    Ожидаемый результат: появится сообщение "Письмо с паролем отправлено на вашу почту, 
    если письма нет — проверьте папку «Спам»." и письмо с паролем придет на вашу почту.

№2  Тестирование входа в аккаунт
    1. Переходим на 21vek.by
    2. Нажимаем кнопку "Войти" в правом верхнем углу.
    3. Появляется форма Входа, на которой вводим свою электронную почту
    и пароль.
    4. Нажимаем кнопку Войти.
    
    Ожидаемый результат: После нажатия кнопки "Войти" вы успешно будете авторизированны 
    под своим аккаунтом.

№3  Тестирование строки поиска товара
    1. Переходим на 21vek.by
    2. Вводим в строку поиска товара "телевизор".
    3. Нажимаем кнопку "Лупа".
    
    Ожидаемый результат: произойдет перенаправление на каталог с телевизорами.

№4  Тестирование фильтрации товаров
    1. Переходим на 21vek.by
    2. Нажимаем на кнопку "Для дома", которая расположена под строкой поиска товара.
    3. Нажимаем "Стиральные машины" в разделе Бытовая техника.
    4. На появившейся странице со стиральными машинами, выбираем чекбокс 
    Atlant в разделе Производители.
    5. Нажимаем кнопку "Показать товары".

    Ожидаемый результат: на странице будут присутствовать только стиральные машины
    производителя Атлант.

№5  Тестирование корзины товаров
    1. Переходим на 21vek.by
    2. Вводим в строку поиска товара "телевизор".
    3. Нажимаем кнопку "Лупа".
    4. Нажимаем кнопку "В корзину", выбрав телевизор, который нам понравился.
    5. Нажимаем кнопку "Корзина" в правом верхнем углу.
    Ожидаемый результат: окажемся на странице с товарами, которые мы выбрали (телевизор).

№6  Тестирование почтовой рассылки о скидках.
    1. Переходим на 21vek.by.
    2. Нажимаеа на кнопку "Скидки, суперцены" в верху страницы.
    3. Нажимаем кнопку "Подписаться на скидки".
    4. Вводим Имя и Электронную почту, на появившейся форме.
    5. Нажимаем кнопку "Отправить".

    Ожидаемый результат: через некоторое время будут приходить письма со скидками
    от данного магазина.

№7  Тестирвоание оформления заказа
    1. Переходим на 21vek.by
    2. Добавляем любой товар в корзину, нажав на кнопку "В корзину" возле товара.
    3. Нажимаем кнопку "Корзину" в правом верхнем углу.
    4. Нажимаем кнопку "Оформить заказ".
    5. Заполяем поля "Адрес доставки" и "Телефон".
    6. Выбираем способ оплаты, нажав на радиобаттон.
    7. Нажимаем кнопку "Подтвердить заказ".

    Ожидаемый результат: звонок менеджера пользователю, чтобы проверить данные заказа.

№8  Ввод названия двух товаров в поисковую строку товаров.
    1. Переходим на 21vek.by
    2. Вводим "телевизор стиральная машина" в поисковую строку.

    Ожидаемый результат: результаты поиска будут содержать список телевизоров, а также  
    стиральных машин.

№9  Просмотр избранных товаров.
    1. Переходим на 21vek.by
    2. Выполняем вход в свой аккаунт.
    3. Нажимаем кнопку "Моё".
    4. В списке выбираем "Просмотренные".

    Ожидаемый результат: на появившейся странице будет список раннее просмотренных вами товаров.

№10 Сравнение товаров
    1. Переходим на 21vek.by
    2. Переходим в раздел, например, Для дома.
    3. Выбираем "Пылесосы".
    4. Нажимаем "Добавить в сравнение" под двумя любыми пылесосами.
    5. Нажимаем "Сравнить товары".

    Ожидаемый результат: на странице появятся характеристики, выбранных нами пылесосов. Произойдет сравнение 
    каждой из характеристик.
