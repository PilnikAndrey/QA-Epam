№1  Тестирование подбора отеля
    1. Переходим на www.agoda.com
    2. Вводим в поле "Введите направление и ..." город.
    3. Выбираем дату заселения.
    4. Выбираем дату отъезда.
    5. Выбираем количество человек, которые будут заселяться, и количество номеров.
    6. Нажимаем кнопку "Искать".
    
    Ожидаемый результат: появится список отелей или квартир, подобранные по нашим параметрам.

№2  Тестирование функции "Семейный режим"
    1. Заполняем данные для поиска жилья.
    2. Нажимаем кнопку "Искать".
    3. На появившейся странице со спсиком жилья, нажимаем кнопку "Семейный режим"
        
    Ожидаемый результат: Список жилья по нашим параметрам обновится. Пояявится список, который будет содержать наименьшую цену жилья и скидки для семей.

№3  Тестирование фильтрации по жилью.
    1. Заполняем данные для поиска жилья.
    2. Нажимаем кнопку "Искать".
    3. В верхней части страницы нажимаем кнопку, например, "Количество звезд".
    4. В списке выбираем 3 звезды.
    
    Ожидаемый результат: список жилья будет состоять только из отелей, которые имеют 3 звезды.

№4  Тестирование сортировки
    1. Заполняем данные для поиска жилья.
    2. Нажимаем кнопку "Искать".
    3. В разделе "Сортировать" выбираем, например, "Расстояние".
    4. Выбираем местоположение, возле которого будет находиться ваш отель.
    
    Ожидаемый результат: список жилья обновится. В верхней части списка будут находиться отели, которые расположены ближе к вашему выбранному месту.

№5  Тестирование функции "Секретные предложения"
    1. Заполняем данные для поиска жилья.
    2. Нажимаем кнопку "Искать".
    3. Нажимаем кнопку "Секретные предложения".
    
    Ожидаемый результат: появится список отелей без его названия. Проживание в этих отелях дешево, т.к. большие скидки. После бронирования, вы узнаете его название.

№6  Тестирование аренды машины
    1. Переходим на www.agoda.com
    2. В toolbar нажимаем "Прокат машин".
    3. В поле "Страна" выбираем страну.
    4. В поле "Город" выбираем город.
    5. В поле "Место" выбираем место.
    6. Выбираем "Дату получения".
    7. Выбираем "Дату возврата".
    8. Нажимаем кнопку "Поиск"

    Ожидаемый результат: появится список автомобилей, их цены и условия аренды.

№7  Тестирвоание поиска трансфера 
    1. Переходим на www.agoda.com
    2. В toolbar нажимаем "Трансфер".
    3. Заполняем поле "Посадка".
    4. Заполняем поле "Высадка".
    5. Заполняем поле "Дата посадки".
    6. Заполняем поле "Время посадки.
    7. Выбираем кол-во пассажиров.
    8. Нажимаем кнопку "Поиск",

    Ожидаемый результат: повится список возможного трансфера: тип авто, цена, время в пути,.

№8  Тестирование бронирования авиабилетов.
    1. Переходим на www.agoda.com
    2. В toolbar нажимаем "Трансфер".
    3. Заполняем поле "Откуда".
    4. Заполняем поле "Куда".
    5. Заполняем поле "Дату вылета".
    6. Выбираем класс.
    7. Указываем количество пассажиров.

    Ожидаемый результат: появится список рейсов по выбранным датам.

№9  Просмотр скидочных предложений.
    1. Переходим на www.agoda.com
    2. В toolbar нажимаем "Текущие предложения".
    
    Ожидаемый результат: появится список предложений со скидками.

№10 Поиск туристических мероприятий
    1. Переходим на www.agoda.com
    2. В toolbar нажимаем "Досуг".
    3. Заполняем поле "Страна".
    4. Заполняем поле "Город".
    5. Выбираем дату мероприятия.
    6. Нажимаем кнопку "Go".

    Ожидаемый результат: появится список туристических мероприятий, экскурсий, а также их цены и продолжительность экскурсии.
